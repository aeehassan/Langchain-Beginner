## Tasks

### Writing Format Translator - translator.py

**Input:** Informal text

```
yo boss, i aint gonna make it 2day. car broke down. c u tmrw.
```

**Output:** Formal text in the form of an email

```
Dear [Manager's Name],

Please accept this email as notification that I will
be unable to attend work today due to unexpected
vehicle trouble. I apologize for the inconvenience
and plan to return to the office tomorrow.

Sincerely, [Your Name]
```

**Use Case:** Corporate settings

### Dungeon Guide Simulator - dungeonguide.py

**Input:** User actions in a dungeon

```
enter dungeon
look around
move forward
attack monster
etc...
```

**Output:** Descriptive responses from the dungeon guide

```
Dungeon Guide: You step into a dimly lit dungeon. The air is damp and the walls are covered in moss. You see a narrow path leading deeper into the darkness.

Dungeon Guide: As you look around, you notice ancient carvings on the walls and hear the distant dripping of water. A faint glow emanates from deeper within the dungeon.
```
